# tin4

Minimal Bun-based Turborepo scaffold modeled after the `create-t3-turbo` workspace layout.

## Structure

```text
apps/
  expo/
  nextjs/
  tanstack-start/
packages/
  api/
  auth/
  db/
  ui/
  validators/
tooling/
  eslint/
  github/
  prettier/
  tailwind/
  typescript/
```

## Commands

- `bun install`
- `bun run build`
- `bun run dev`
- `bun run lint`
- `bun run typecheck`
